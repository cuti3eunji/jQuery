dom 객체와 jquery 객체 반환

jquery를 dom으로 반환 : jquery객체.get[N] / jquery객체[N]
dom 객체를 jquery객체로 반환 : $(dom 객체 )

jq = $('h1');
jq.css('color', 'blue');


-jquery 객체를 dom으로 변환
dom = jq.get(2); //jq[2]
dom.style.color = 'red';
-dom 객체를 jquery로 변환
$(dom).css('color','green');
-전체 h1 중에서 세번째 선택 : jquery
jq3 = $('h1:eq(3)') // h1 선택자 사용시 $ 사용
jq3.css('color', 'hotpink');
